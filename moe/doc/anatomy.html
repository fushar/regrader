The Anatomy of Moe

Moe lives in the following directory structure:

  • box/ – the sandbox module
  • build/ – scripts used for compilatuon
  • doc/ – this documentation
  • eval/ – the evaluator (grader)
  • examples/ – example problems
  • judge/ – judging helpers
  • maint/ – scripts used by Moe's maintainers
  • mop/ – various stuff for the MO-P contest
  • sherlock/ – libsherlock, used by the submit server
  • submit/ – the submit server
  • ucw/ – libucw, used by the submit server
  • utils/ – various utilities

When installed, it will create the following hierarchy:

  • eval – all data belonging to the evaluator, inaccessible to contestants
  • public – public data made available to the contestants
  • users – home directories of the contestants
